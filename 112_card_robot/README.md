# 112 Card Robot

Python-робот для автоматического принятия карточек 112 через пользовательский интерфейс сайта.

Робот не использует координатный кликер. Он управляет браузером через Playwright, ищет карточки в первом столбце по CSS-селектору, кликает по первой найденной карточке, ведет журнал, делает скриншоты ошибок и пытается восстановить работу после сбоя страницы или разлогина.

## Что делает робот

- открывает страницу с карточками;
- проверяет, не появилась ли страница авторизации;
- авторизуется при необходимости;
- ищет карточки в первом столбце;
- кликает по карточке, что считается фактом принятия;
- записывает результат в `logs/robot.log` и `logs/events.csv`;
- делает скриншот при ошибке;
- перезагружает страницу при сбое;
- может отправлять Telegram-уведомления при критических ошибках.

## Установка

```bat
cd C:\112_robot
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
playwright install chromium
```

## Настройка

Скопировать пример конфига:

```bat
copy config.example.json config.json
```

Заполнить `config.json`.

Главное поле — `new_cards_selector`. Его нужно взять на рабочем месте через F12: выбрать карточку в первом столбце, затем Copy selector.

## Запуск

```bat
python -m src.robot
```

Или через файл:

```bat
run_robot.bat
```

## Боевой принцип

Робот должен запускаться на отдельном ПК или виртуальной машине. Не рекомендуется запускать его на рабочем месте оператора, где параллельно вручную работают с тем же сайтом.

Робот только принимает карточку фактом клика. Он не закрывает обращение, не меняет содержание карточки и не принимает решение по дальнейшей обработке.
